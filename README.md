# direct-desk-solutions

Single-page demo mockup for Direct Desk Solutions — a UK office furniture retailer. Eight screens (Home, New, Used, Product, Basket, Checkout, CRM, CRM Upload) in one self-contained `index.html` using React + Tailwind via CDN.

**Live demo:** https://roam-agency.github.io/direct-desk-solutions/

## Local preview

Open `index.html` directly in a browser — no build step.

## Deploy

`main` auto-deploys to GitHub Pages via `.github/workflows/pages.yml`.
