# Direct Desk Solutions

E-commerce site and admin CRM for Direct Desk Solutions, a UK office furniture retailer (new + second-hand). Currently scaffolding the production app.

## Stack (in build)

- Next.js 15 (App Router) + TypeScript + Tailwind CSS
- Supabase (Postgres database + auth)
- Cloudinary (image storage and CDN)
- Stripe (payments)
- Resend (transactional email)
- Netlify (hosting)

## Mockup files

`index.html` and `app.jsx` at the repo root are the original design prototype (React + Tailwind via CDN). These will be removed once the production app supersedes them.
